# 🍉 Juice-store/ Site de venda de sucos

- Projeto básico simulação site de vendas de suco 
- Feito com HTML, CSS e JavaScript com recursos do <a href="https://getbootstrap.com/">BootStrap</a>

<img src="https://user-images.githubusercontent.com/95694730/175106199-d9d03f1a-6cd3-489c-a679-abc65eed30ff.png" style="width: 800px;">

## ✅ Funcionalidades
### 🔸 Selecione o produto desejado
- Clique em "Comprar agora" para ser direcionado para página de compra
<img src="https://user-images.githubusercontent.com/95694730/180294551-834f810e-ead5-416d-9eb2-38212e50eb22.png" style="width: 900px;">

### 🔸 Selecione os dados do produto desejado
- Selecione a quantidade, e o tamanho do produto
<img src="https://user-images.githubusercontent.com/95694730/175106390-fb0be328-96e5-4122-aa5d-0c1bc62d41c2.png" style="width: 800px;">
- Após colocar os dados e clicar "Colocar na sacola" aparecerá o valor da compra
<img src="https://user-images.githubusercontent.com/95694730/180297256-4dc8ae65-0c00-4e88-a095-8c259a6755cc.png" style="width: 800px;">
